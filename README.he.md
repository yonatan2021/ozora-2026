# פסטיבל אוזורה 2026 - עוזר לוח הופעות דינמי

**עברית 🇮🇱** | [English 🇬🇧](README.md)

![לוח הופעות פסטיבל אוזורה 2026](./src/assets/ozora_banner.png)

ברוכים הבאים ל**עוזר לוח ההופעות האינטראקטיבי של פסטיבל אוזורה 2026** (Ozora Festival 2026) — אפליקציית ווב מתקדמת, מעוצבת ועשירה בפיצ'רים שנועדה לעזור למשתתפי הפסטיבל האגדי בהונגריה (25 ביולי - 3 באוגוסט 2026) לנהל את הזמן שלהם בצורה הטובה ביותר.

אפליקציה זו נבנתה לא רק ככלי עזר פשוט להצגת שעות הופעה, אלא כחוויה חושית מלאה המעבירה את הרוח הפסיכדלית, הקוסמית והטבעית של פסטיבל אוזורה ישירות לדפדפן. הממשק עבר אופטימיזציה מלאה לתנאי פסטיבל אמיתיים בשטח: קריא תחת אור שמש ישיר וחזק, אטמוספרי ומותאם לחשיכה מוחלטת בלילה, ונוח לשימוש מהיר בטלפון הנייד בתנאי קליטה קשים.

---

## 🌟 תכונות מרכזיות

### 1. 🌀 רקע פסיכדלי אינטראקטיבי ונושם
*   **תנועה אורגנית וטבעית:** הרקע של האפליקציה כולל אלמנטים גיאומטריים ופסיכדליים מבוססי SVG ו-Canvas שמגיבים בעדינות למחוות מגע ותזוזת העכבר. התנועה מרגישה נושמת וזורמת אך אינה מפריעה לקריאות של המידע.
*   **מחזור נושאים דינמי (יום/לילה):** האפליקציה מזהה את השעה הנוכחית (או השעה המדומה בסימולטור) ומעדכנת אוטומטית את העיצוב לאחד מארבעה מצבים:
    *   🌅 **זריחה** (05:00 - 07:00): גווני פסטל חמימים המדמים את עלות השחר.
    *   ☀️ **יום** (07:00 - 18:00): ניגודיות גבוהה במיוחד (High Contrast) שנועדה לאפשר קריאה קלה תחת השמש הקופחת.
    *   🌇 **שקיעה** (18:00 - 20:00): זוהר מטאפיזי בגוונים כתומים, אדומים וזהובים של שעת בין הערביים.
    *   🌌 **לילה** (20:00 - 05:00): מצב כהה עמוק, קוסמי ועשיר המשלב גיאומטריה מקודשת, אורות ניאון זוהרים ופאנלים שקופים דמויי זכוכית (Glassmorphism).

### 2. ⏱️ סימולטור זמן מתקדם ומצבי הופעות בזמן אמת
*   **סליידר מכונת זמן:** סליידר אינטראקטיבי המאפשר למשתמש לגרור את הזמן קדימה ואחורה כדי לראות בדיוק אילו במות פעילות בכל יום ובכל שעה במהלך הפסטיבל.
*   **מה מנגן עכשיו? (Live Status):** מודל ייעודי המרכז במקום אחד את כל האמנים שמנגנים **עכשיו** (Now Playing) ואת אלו שיעלו **מיד אחריהם** (Next Up) בכל אחת מהבמות השונות בפסטיבל.
*   **חיווי סטטוס דינמי:** המערכת מחשבת אוטומטית אם סט נמצא בעיצומו (`Active`), עתיד להתחיל בקרוב (`Future`) או כבר הסתיים (`Past`).

### 3. 📊 גריד מערכת שעות אופטימלי למחשבים וטאבלטים
*   **סינון במות ריקות:** כדי למנוע גלילה מיותרת ומעייפת, האפליקציה מזהה באילו במות אין הופעות ביום שנבחר ומעלימה אותן מהתצוגה בצורה חכמה.
*   **כיווץ מרווחי זמן:** מרווחי שעות ארוכים ללא הופעות מכווצים אוטומטית כדי לאפשר סריקה מהירה של הלוח.
*   **שיוך צבע ייחודי לבמות:** לכל במה בפסטיבל (Ozora Stage, Pumpui, The Dome, Dragon Nest, Visium Garden, Tek Zero) יש סכמת צבעים ואישיות ויזואלית קבועה שעוזרת להתמצאות מהירה.

### 4. 📱 פיד כרונולוגי ייעודי לטלפונים ניידים
*   **עיצוב Mobile-First:** במכשירים ניידים, הגריד הרחב הופך אוטומטית לפיד אנכי כרונולוגי המותאם לשימוש ביד אחת בשטח הפסטיבל.
*   **ניווט מהיר בין במות:** סינון מהיר באמצעות כפתורי פילטר אופקיים המאפשרים לבודד במה ספציפית ברגע.
*   **מקשי מגע מוגדלים:** אזורי הלחיצה הוגדלו כדי להבטיח שימוש נוח גם תוך כדי הליכה או בתנאי תאורה מורכבים.

### 5. ⭐️ לוח הופעות אישי מותאם במיוחד (הלוח שלי)
*   **שמירה מקומית (localStorage):** סימון אמנים מועדפים בכוכב כדי ליצור לוח זמנים מותאם אישית שנשמר בדפדפן ומלווה אותך גם אחרי ריענון העמוד או סגירתו.
*   **מפתחות שמירה יציבים:** מעבר לשימוש במפתח מורכב (`${artist}::${stage}::${date}::${start}`) במקום אינדקסים משתנים של מערך, מה שמונע קריסה או שיבוש של המועדפים שלך במקרה של עדכון קובץ השעות המרכזי.
*   **שיתוף הלוח לחברים:** כפתור **שתף לוח זמנים** מייצר קישור מותאם אישית הכולל את האמנים שבחרת. כאשר חבר פותח את הקישור במכשיר שלו, המועדפים שלך ימוזגו אוטומטית עם שלו ותוצג הודעת חיווי (Toast).

### 6. 📴 עובד אופליין — ללא צורך באינטרנט (PWA)
*   **תמיכה מלאה במצב אופליין:** האפליקציה בנויה כ-Progressive Web App (PWA) עם Service Worker שמאחסן את כל הנכסים ונתוני לוח ההופעות בביקור הראשון. חשוב לפתוח אותה פעם אחת עם אינטרנט לפני הפסטיבל; אחרי הטעינה הראשונה היא עובדת גם במצב טיסה — מותאמת במיוחד לפסטיבלים עם קליטה אפסית.
*   **התקנה למסך הבית:** באנר התקנה דו-לשוני מעודד את המשתמשים להוסיף את האפליקציה למסך הבית לפני הפסטיבל, כך שתתנהג כמו אפליקציה מקורית עם זמני טעינה מיידיים.

### 7. 🎯 ניהול לוח חכם
*   **סימון עדיפויות:** מעבר בין רמות עדיפות לכל הופעה מועדפת — **חובה לראות** (אייקון אש), **רוצה לראות** (כוכב), או **אולי** (עמום) — כדי לתכנן את ימי הפסטיבל בבהירות.
*   **הערות אישיות:** הוספת הערות קצרות לכל הופעה (לדוגמה: "להיפגש עם דני פה", "להביא מים"). ההערות מוצגות על כרטיסי ציר הזמן ונשמרות מקומית.
*   **זיהוי קונפליקטים:** האפליקציה מזהה אוטומטית כשתי הופעות מועדפות חופפות בזמן ומסמנת אותן עם תג אזהרה המציג את משך החפיפה.

### 8. 📲 שיתוף QR וייצוא תמונת לוח
*   **החלפת קוד QR:** יצירת קוד QR של הלוח האישי שלך. חברים סורקים אותו עם המצלמה — בלי צורך באינטרנט. נתוני הלוח מקודדים ישירות ב-URL, מה שהופך את זה למושלם לשיתוף טלפון-לטלפון בפסטיבל.
*   **ייצוא לוח כתמונה:** ייצוא הלוח האישי כולו כתמונת PNG יפה וניתנת לשיתוף (מותאמת לסטוריז באינסטגרם/וואטסאפ). התמונה נוצרת בצד הלקוח באמצעות `html2canvas` — ללא צורך בשרת או חיבור לאינטרנט.
*   **השוואת לוחות חברים:** ייבוא לוח של חבר (דרך QR או לינק) ושמירתו בנפרד עם השם שלו. השוואת הבחירות שלכם זה לצד זה: ראו מועדפים משותפים ("שניכם בחרתם"), גלו מה החבר בחר שאתם לא, ותאמו את חוויית הפסטיבל שלכם ביחד.

### 9. 🔍 מנוע חיפוש חכם בזמן אמת
*   **חיפוש מיידי:** חיפוש מהיר לפי שם אמן, סוג מוזיקה, סוג סט או במה.
*   **מיקוד והדגשה:** לחיצה על תוצאת חיפוש מעבירה את המשתמש מיידית ליום ולבמה המתאימים, גוללת בצורה חלקה (Smooth Scroll) ישירות לכרטיסיית ההופעה ומפעילה אפקט הבהוב זוהר זמני כדי למקד את העין.

### 10. 🇮🇱 תאימות מלאה לחוקי הגנת הפרטיות בישראל (GA4 Consent)
*   **חסימת מעקב ברירת מחדל:** מנגנון מובנה המונע טעינה של סקריפטים של גוגל אנליטיקס (GA4) עד לקבלת אישור מפורש מהמשתמש.
*   **ניהול העדפות עוגיות:** פאנל הגדרות פרטיות מעוצב המאפשר למשתמש לבחור בצורה פרטנית אילו קבצי עוגיות לאשר (עוגיות חיוניות, סטטיסטיקה, פונקציונליות ושיווק).

## 🛠️ פרטים טכניים וארכיטקטורה

### טכנולוגיות
*   **פריימוורק:** React 18+ עם כלי הבנייה המהיר Vite.
*   **אייקונים:** Lucide React.
*   **עיצוב וסטיילינג:** Vanilla CSS טהור ומותאם אישית. שימוש אינטנסיבי במשתני HSL מאפשר לשנות את האווירה הויזואלית של כל האפליקציה על ידי החלפת קלאס אחד על גבי קונטיינר העל.
*   **בסיס נתונים:** המידע כולו מנוהל באמצעות קובץ JSON מובנה ומסודר (`timetable.json`).

### 1. מערכת ערכות הנושא ב-CSS
החלפת הצבעים והאפקטים מתבצעת על ידי הגדרת משתנים בצורה הבאה:

```css
/* דוגמה להגדרת משתני עיצוב ב-index.css */
.app-container.theme-day {
  --bg-color: #f7f3e8;
  --text-primary: #12100e;
  --glow-intensity: 0;
  --blur-amount: 0px;
}

.app-container.theme-night {
  --bg-color: #0b0713;
  --text-primary: #f0e6ff;
  --glow-intensity: 1.5;
  --blur-amount: 12px;
}
```

### 2. יצירת מפתח מזהה ייחודי להופעות
כדי לשמור על המועדפים שלך יציבים גם לאחר שינויים במאגר המידע:

```javascript
// מתוך src/utils/time.js
export function getSetUniqueKey(set) {
  return `${set.artist}::${set.stage}::${set.date}::${set.start}`;
}
```

### 3. מנגנון הסכמת עוגיות (Cookie Consent)
חסימת גוגל אנליטיקס כברירת מחדל ושליטה במצב ההסכמה:

```javascript
// מתוך src/utils/consent.js
window.gtag('consent', 'default', {
  'analytics_storage': analyticsGranted ? 'granted' : 'denied',
  'ad_storage': marketingGranted ? 'granted' : 'denied',
  'security_storage': 'granted' // תמיד מאושר כיוון שהכרחי
});
```

## 🚀 הוראות הפעלה ופיתוח מקומי

על מנת להריץ את הפרויקט על המחשב האישי שלך, בצע את הצעדים הבאים:

### דרישות קדם
*   Node.js (גרסה 18 ומעלה מומלצת)
*   npm (גרסה 9 ומעלה)

### התקנה
1. שכפל את מאגר הקוד (Clone) למחשב המקומי.
2. התקן את התלויות באמצעות הרצת הפקודה הבאה בטרמינל:
   ```bash
   npm install
   ```

### הרצה מקומית במצב פיתוח
להפעלת שרת פיתוח מקומי עם ריענון חם (HMR):
   ```bash
   npm run dev
   ```
פתח את הדפדפן בכתובת [http://localhost:5173](http://localhost:5173) כדי לצפות באפליקציה.

### בנייה לגרסת ייצור (Production)
ליצירת תיקיית בנייה מותאמת ואופטימלית להעלאה לשרת (`dist/`):
   ```bash
   npm run build
   ```
