# מדריך לכתיבת תוכן — Festival Guides

## איך מוסיפים קטגוריה חדשה

1. צרו קובץ חדש בתיקייה `content/guides/` עם סיומת `.md`
   - לדוגמה: `food.md`, `safety.md`, `first-timer.md`
2. בראש הקובץ, הוסיפו את ה-frontmatter (המטא-דאטה):

```yaml
---
title: שם הקטגוריה
icon: compass
order: 5
---
```

3. כתבו את התוכן מתחת ל-frontmatter

## איך מוסיפים נושא תחת קטגוריה

הוסיפו כותרת `##` (שני סולמיות + רווח) בתוך הקובץ:

```markdown
## שם הנושא

התוכן של הנושא כאן...
```

כל `##` = נושא נפרד שמופיע כאקורדיון באתר.

## Frontmatter — שדות

| שדה | חובה? | מה זה | ברירת מחדל |
|------|--------|--------|-----------|
| `title` | כן | שם הקטגוריה שמוצג באתר | — |
| `icon` | לא | אייקון שמוצג בכרטיס | `compass` |
| `order` | לא | סדר הצגה (1 = ראשון) | 999 |

## רשימת אייקונים זמינים

| שם ב-frontmatter | מתאים ל... |
|-------------------|------------|
| `tent` | קמפינג, לינה |
| `backpack` | ציוד, ארגון |
| `compass` | ניווט, התמצאות |
| `map` | תחבורה, הגעה |
| `utensils` | אוכל, מסעדות |
| `droplets` | מים, שתייה |
| `sun` | מזג אוויר, הגנה |
| `heart` | בריאות, רווחה |
| `shield` | ביטחון, בטיחות |
| `music` | מוזיקה, במות |
| `users` | קהילה, חברתי |
| `phone` | תקשורת, קישוריות |

| `zap` | חשמל, טעינה |

## איך order עובד

- מספר נמוך = מוצג ראשון
- `order: 1` יהיה הכרטיס הראשון, `order: 2` השני, וכו׳
- אם לא מציינים order, הקטגוריה תופיע בסוף
- אפשר להשתמש בכל מספר (לא חייב רצף: 1, 5, 10 — זה בסדר)

## עיצוב טקסט

- **מודגש**: `**טקסט מודגש**`
- רשימה: שורה שמתחילה ב-`- ` (מינוס + רווח)
- קישור: `[טקסט](https://example.com)`
- פסקאות: שורה ריקה בין פסקאות

## דוגמה מלאה

```markdown
---
title: אוכל ומים
icon: utensils
order: 5
---

## מים

לפחות 3 ליטר ביום. יש נקודות מילוי ברחבי הפסטיבל.

## אוכל בשטח

יש דוכנים עם אוכל טבעוני, הונגרי, ופלאפל.
מחירים סבירים. אפשר גם להביא כיריים גז קטנות.

## טיפים
- תביאו חטיפים מהבית — **חוסך כסף**
- דוכן הפלאפל ליד הדום — מומלץ
```
